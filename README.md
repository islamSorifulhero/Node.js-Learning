## ⚙️ Installation

Move into the project directory:

```bash
cd Node-js-Learning
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## 📌 Useful Commands

### Initialize Node Project

```bash
npm init -y
```

### Install Express

```bash
npm install express
```

### Install Development Dependencies

```bash
npm install -D nodemon
```

### Start Application

```bash
npm start
```

### Development Mode

```bash
npm run dev
```
